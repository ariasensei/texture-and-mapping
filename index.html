Texture and Mapping

18-Week Course Index — Principles, Techniques & Industry Practices

Course Description

This course covers the fundamentals and advanced techniques of texture creation and UV mapping for 3D assets. Students will learn industry-standard workflows using Blender and Substance Painter, progressing from basic color theory through PBR material creation to final portfolio-ready game and film assets.

Instructor: MKDB

🎯 Learning Outcomes

• Create production-ready PBR textures • Master UV mapping and optimization • Build a professional texture portfolio • Apply textures for games and film • Understand procedural and hand-painted workflows

🖥️ Software & Tools

• Blender 4.x • Adobe Substance Painter • Adobe Photoshop • Marmoset Toolbag (viewing) • Pureref (reference boards)

📋 Requirements

• Laptop/PC with dedicated GPU • Basic 3D modeling knowledge • Completed 3D Modeling 1 course • External mouse recommended • Minimum 8GB RAM
